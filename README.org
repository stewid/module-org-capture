* org-capture-module

  Capture notes in org-mode from the Evolution personal information
  management application.

** Installation

   #+BEGIN_SRC sh
   git clone https://github.com/stewid/org-capture-module.git
   cd org-capture-module
   mkdir _build
   cd _build
   cmake -G "Unix Makefiles" ..
   make
   make install
   #+END_SRC
